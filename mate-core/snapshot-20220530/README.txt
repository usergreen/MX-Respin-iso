This ISO image offers MATE Desktop Environment with Core components only,
which supports Multilingual with Japanese IME.
Besides, Firefox browser and Synaptic Package Manager are shipped.

Supported locale options:
de_DE.UTF-8 - German locale for Germany
en_US.UTF-8 - English locale for the USA
es_ES.UTF-8 - Spanish locale for Spain
fr_FR.UTF-8 - French locale for France
it_IT.UTF-8 - Italian locale for Italy
ja_JP.UTF-8 - Japanese language locale for Japan
nl_NL.UTF-8 - Dutch locale for the Netherlands
nn_NO.UTF-8 - Nynorsk language locale for Norway
pt_PT.UTF-8 - Portuguese locale for Portugal
ru_RU.UTF-8 - Russian locale for Russia
zh_CN.UTF-8 - Chinese locale for Peoples Republic of China

Note:
No default language (locale) is set; that is "C" for English. Click on the upper right corner on the login screen, then choose a locale from a drop-down list. Simply logout and then login again can change a locale for Desktop session.

* In order to enable Japanese input, click IME toolbar which appears at the right corner on the bottom. Then select item "Anthy" or "Mozc". Now you can start Japanese input by pressing Shift + Space key, or alternatively "半/全 漢字" key if your keyboard is Japanese layout. To return to previous mode, just press the same key.
